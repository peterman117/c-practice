## 今日学习内容：指针

### 1. 什么叫指针？
指针就是一个**存放内存地址**的变量。以前完全搞不懂，今天画了一下内存图，好像清晰一些了。

### 2. 写了一段测试代码
下面是我今天在VS里写的代码，用来测试指针的基本用法：

```c
#include <stdio.h>

int main() {
    int num = 10;
    int *p = &num;  // p是一个指针，存了num的地址
    
    printf("num的值是：%d\n", num);
    printf("p指向的地址是：%p\n", p);
    printf("p指向的值是：%d\n", *p);
    
    return 0;
}
